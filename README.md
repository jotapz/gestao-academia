# 🏋️ Gym Management System (Backend)

![Status](https://img.shields.io/badge/status-completed-brightgreen)

A production-ready Backend solution designed to digitize fitness center operations. This system manages complex relational data between members, professional staff, and workout scheduling, ensuring data integrity and high performance through a structured relational architecture.

---

### 🔍 Project Overview
Developed to solve the logistical challenges of managing high-traffic fitness centers, this system centralizes member data and training oversight. It bridges the gap between administrative needs and student progress, providing a reliable API for gym automation and record-keeping.

---

### ✨ Core Features

- 👥 **Student Lifecycle Management:** Full CRUD implementation for member registration, profile updates, and status monitoring.
- 👨‍🏫 **Instructor Oversight:** Dedicated module for managing professional staff, credentials, and specialties.
- 📋 **Relational Workout Mapping:** Specialized logic that links customized training routines to students, moderated by assigned instructors.
- 📊 **Attendance & Frequency Tracking:** Automated logs for session monitoring, allowing for accurate frequency analytics.

---

### 🚀 Tech Stack

- **Backend Logic:** - **Java 22:** Utilizing modern Java features for clean, efficient, and robust code execution.
- **Data Persistence:** - **MySQL 8:** Optimized relational database ensuring ACID compliance and data consistency.
- **Engineering Tools:** - **Git & GitHub:** Version control and collaborative workflow management.
  - **MySQL Workbench:** Used for advanced Entity-Relationship Diagram (ERD) design and database modeling.
